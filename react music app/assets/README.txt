Add these files for Expo to run without errors:
- icon.png (e.g. 1024x1024)
- splash.png
- adaptive-icon.png (e.g. 1024x1024)

You can copy them from a new Expo app:
  npx create-expo-app@latest temp-app --template blank-typescript
  then copy icon.png, splash.png, adaptive-icon.png from temp-app/assets/ into this folder.
